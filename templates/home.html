NES, SNES, N64 Search - Web App

This is a RESTFUL app to search for videos games



HTML VERSION

Click here for a web search


API (JSON) VERSION

Available methods:

1. /search/?format=json
Parameters (POST request):
         query: The keyword you are looking for

2. /game/GAME_ID/?format=json
Parameters (GET request):
         GAME_ID: The ID of the game. You can know this from the search results.
         Example: http://54.68.197.149:8080/game/544/?format=json
